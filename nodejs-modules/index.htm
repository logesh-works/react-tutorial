React.js Components

Summary: in this tutorial, you will learn about React.js Components and understand how they work.

What are React Components?

React components are the building blocks of a React application. They are reusable pieces of code that encapsulate a part of the user interface. React applications are typically composed of multiple components that work together to create complex UIs.

Functional Components

Functional components are JavaScript functions that return JSX (JavaScript XML) to describe the UI. They are also known as stateless components. Here's an example:

        
function Greeting(props) {
  return <h1>Hello, {props.name}!</h1>;
}

Class Components

Class components are JavaScript classes that extend React.Component and use a render method to return JSX. They can also have state and lifecycle methods. Here's an example:

        
class Counter extends React.Component {
  constructor(props) {
    super(props);
    this.state = { count: 0 };
  }

  render() {
    return <p>Count: {this.state.count}</p>;
  }
}

Creating a React App

Before you can work with React components, you'll need to set up a React application. You can use create-react-app to quickly create a new React project:

        
# Install create-react-app globally (if not already installed)
npm install -g create-react-app

# Create a new React app
create-react-app my-react-app
cd my-react-app

Using React Components

Now, let's see how to create and use React components in your application.

Create a Component

Create a new component in a separate .js file. For example, create a Greeting.js file with the following content:

        
import React from 'react';

function Greeting(props) {
  return <h1>Hello, {props.name}!</h1>;
}

export default Greeting;

Use the Component

In your main application file (usually App.js), you can import and use the component:

        
import React from 'react';
import Greeting from './Greeting'; // Import your component

function App() {
  return (
    <div>
      <Greeting name="John" /> {/* Use the component */}
    </div>
  );
}

export default App;

Rendering Components

To render your React app, use ReactDOM.render() in your index.js file:

        
import React from 'react';
import ReactDOM from 'react-dom';
import App from './App';

ReactDOM.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
  document.getElementById('root')
);

Running Your App

Start your development server to run the React app:

        
# Navigate to your project directory
cd my-react-app

# Start the development server
npm start

This is a very basic introduction to React components. React offers more advanced features like props, state, lifecycle methods, and hooks, which allow you to build complex and interactive user interfaces. You can explore these concepts in more detail as you become more comfortable with React.
